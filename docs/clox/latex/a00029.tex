\doxysubsection{Detailed Description}
\mbox{\hyperlink{a00029}{Chunk}} Data Structure. 

Represents a dynamic array, which will be automatically sized as data is added.

Holds the\+:
\begin{DoxyItemize}
\item {\ttfamily count}\+: Number of bytes within the chunk.
\item {\ttfamily capacity}\+: Spaces available within the block of memory reserved for this chunk.
\item {\ttfamily code}\+: The bytecode contained within the chunk. 
\end{DoxyItemize}
